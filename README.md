# Friend Request Card

A simple mini project using **HTML**, **CSS**, and **JavaScript** that allows users to toggle a "Friend" status.  
Clicking **Add Friend** changes the status to green **Friends**,  
and clicking **Remove** changes the status back to red **Stranger**.
